# PH Estate Manager V2

## Stage 6

Includes:
- Projects module
- Plots module
- Clients module
- Sellers module
- Alphabetical sorting rules
- Plot sorting by project name, then natural plot number
- Auto dash formatting for plot numbers, CNIC, and Pakistani phone numbers
- Urdu auto-fill helper buttons for name, father name, and address fields

Note: Urdu auto-fill uses an online translation endpoint for prototype testing. For final office use, this should be replaced with a stable API or Supabase Edge Function.
